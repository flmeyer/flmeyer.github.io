A self-contained LaTeX document class (`cv.cls`) for building CVs and DIN 5008-compliant cover letters with a shared, configurable look. It uses TikZ to render the signature design touches — a curved header banner, a circular profile photo, gradient section dividers, and a timeline of career stations.

The class is bilingual (English/German) and exposes key-value options for accent color, page frames, section-header style, and signatures, so a complete, visually consistent application can be assembled by filling in a few fields.
