openDSME is a portable, open-source implementation of the IEEE 802.15.4 DSME MAC layer, bringing TDMA guaranteed time slots and multi-channel hopping to low-power wireless for deterministic, multi-hop industrial networks. The same C++ core runs in OMNeT++/INET simulation and on RIOT OS hardware.

As a core contributor I built enhancements within the DSME standard and beyond — slot allocation, CAP reduction, group acknowledgements, and multi-hop scheduling — detailed in several [publications](/publications/).
